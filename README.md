# 자취왕 백종원

자취왕 백종원은 경희대학교 2018학년도 2학기 데이터베이스 수업 중에 만들어진 서비스로서, 가지고 있는 재료를 말하면, 그 재료에 맞춰 요리할 수 있는 음식의 레시피를 제공해 주는 서비스입니다.

## 사용 기술 스택

사용 기술 스택은 다음과 같습니다.
+ MySQL
+ Node.js
+ jQuery

## Installation
필요한 모듈 설치 방법은 다음과 같습니다.

```bash
$ npm install
```

## Usage

서버 실행법은 다음과 같습니다.

```bash
$ node app.js
```
